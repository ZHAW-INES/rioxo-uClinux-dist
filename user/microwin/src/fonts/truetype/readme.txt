Microsoft free fonts
====================
See readme.ms, must include entire distribution according
to license agreement, including fonts.dir* and encodings.dir.
These fonts, although large, have hint information allowing
their use at smaller point sizes.  Most browsers run quite
well with Arial, Times Roman and Courier New typefaces.

Omega Project free fonts
========================
See readme.omega.  These fonts are all Roman Serif fonts,
available in five encodings, ISO8859-1 (Latin-1) through
ISO8859-5 (Latin-5).  A Vietnamese VISCII font is also
included.  Each font comes in regular, bold, italic and
bold italic.  These fonts are less than 30% of the size
of the Microsoft fonts, but don't contain the hint
information for smaller point sizes, so they look better
at about 18 pt size.
